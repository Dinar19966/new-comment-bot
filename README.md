

## Project setup
## Установи все библиотеки перед запуском

```bash
$ npm install
```
## Создай и заполни .env файл
```bash
cp .env.example .env
```
## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
